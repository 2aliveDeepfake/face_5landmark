# face_5landmark
얼굴 5개 랜드마크 표시하는 코드

test_file.py에 있는 parser에 기능이 설명되어 있음 

사용할 수 있는 기능 
1. 얼굴에 5개 포인트 점찍어서 저장
2. 프레임에서 얼굴 detect한 후 크롭해서 저장
3. 얼굴 크롭한 이미지에서 이마, 눈, 오른쪽눈, 왼쪽눈, 코, 입, 턱 잘라서 저장 

사용하고 싶은 기능은 defalt=True, 사용하고 싶지 않은 기능은 defalt=False로 해놓고 실행하면 됨
